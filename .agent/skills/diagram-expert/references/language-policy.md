# Diagram Language Policy

Use this reference when deciding language and output style.

## Language Rules

- Respond in Thai by default
- Use Case diagrams: actor and use case names should be Thai-first
- Class / Sequence / ER diagrams: identifiers that must match code stay in English
- Never localize code identifiers

## Output Rules

- Use Case -> tables only
- Class / Sequence / ER -> Mermaid plus semantic tables
- Never output code alone without an explanatory table
- If evidence is insufficient, state the limitation clearly
